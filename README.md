# 🌿 PollenPal

Pollen forecast & allergy safety score — breathe easier.

**[Try it →](https://magic-dev-kz.github.io/pollenpal/)**

## Features
- Daily pollen safety score (grass, tree, weed)
- 7-day pollen calendar at a glance
- Medication reminder notifications
- Location-based forecast via Open-Meteo API
- Dark / light mode

## Why PollenPal?
Allergy sufferers check the weather but rarely get actionable pollen data. PollenPal turns raw pollen counts into a simple safety score and a weekly outlook — plus reminds you to take your meds before symptoms hit.

## Tech
- Single HTML file — nothing to install
- Works offline (PWA)
- No accounts, no tracking
- Your data stays in your browser (localStorage)
- Free forever
